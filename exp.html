Testing Infrastructure

Clone our testing infrastructure as follows: 

.../ae $ git clone https://github.com/plum-umd/adaptive-concretization.git

This infrastructure is tested under Python 2.7.x. If you are using Python 3.2/3.3, replace the usage of subprocess32 at line 8 in psketch.py with subprocess. If you are using Python 2.x and want to examine the back-end behavior for various randdegree, which corresponds to Table 1 in the paper, you need to install subprocess32. If you are not using Python 2.7.9 or Python 3.4 and later, you need to install pip first.

On Debian and Ubuntu, 

$ sudo apt-get install python-pip

On Mac OS X, if you prefer to use macport, 

$ sudo port install py-pip

On other OSes, download get-pip.py and run the following (which may require administrator access): 

$ (sudo) python get-pip.py

Once pip is ready, install subprocess32: 

$ pip install subprocess32
